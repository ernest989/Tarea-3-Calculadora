--Intrucciones de uso--
1.- Para usar la calculadora es necesario compilar desde terminal, para ello es necesario abrir una terminal e ir a la carpeta src, estando allí hay que utilizar el siguiente comando:
	javac Calculadora.java

2.- Una vez compilado el código se puede ejecutar, para ello desde la misma terminal y la misma carpeta src hay que introducir el siguiente código:
	java Calculadora

3.- Se abrirá un cuadro de texto que describe las operaciones que maneja la calculadora. Sean a y b números reales, y donde +,-,*,/,s,c,t,r,() son operadores, las posibles operaciones son:
	suma de a más b            -> a+b
	resta de a menos b 	       -> a-b
	multiplicación de a por b  -> a*b
	división de a entre b      -> a/b
	raiz cuadrada de a         -> r a
	seno de a                  -> s a
	coseno de a                -> c a
	tangente de a              -> t a
Si deseas introducir una expresión aritmetica con parentesis recuerda que parentesis que abres parentesis que cierras.
Ten en cuenta que la calculadora toma el operando de las funciones trigonométicas en radianes.

4.- Para salir del programa solo escribe "exit" desde la ventana del mismo, cierra la terminal o utiliza ctrl+c.
